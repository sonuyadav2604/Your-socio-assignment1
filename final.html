Injury Tips
Hamstring Strain
Apply ice for 20 mins every 2 hours in first 48 hours. Avoid stretching immediately. Gentle walking is okay if pain is 3/10 or less. Consult if pain is severe.
Knee Pain (Patellar)
Rest from jumping activities. Strengthen VMO with quad sets. Foam roll quad and IT band daily. Avoid squatting below 90°.

Storage Architecture
Bucket: test-videos → stores all raw MP4/MOV files
Path format: players/{player_uuid}/tests/{year}/{month}/{test_video_uuid}.mp4
Bucket: thumbnails → auto-generated frame extracted via serverless function on upload
Access Policy: Players can only read/write their own path. Coaches get read access to assigned player paths only.
Offline: Videos recorded offline saved to local IndexedDB / device storage. Sync queue picks up on network reconnect.
